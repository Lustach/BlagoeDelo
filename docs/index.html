Помогаем Церкви с помощью современных технологий

Цель проекта

В современном мире все используют безналичные методы оплаты. Это означает проблемы для православных организаций, которые зависят от помощи жертвователей.

БЛАГОЕ ДЕЛО создает комплексную инфраструктуру решений для того, чтобы поддержать православные приходы по всей России.

Используя свою технологическую экспертизу, мы создали инструменты, которые помогут Церкви сохранить и приумножить необходимый ей поток пожертвований, а прихожанам чувствовать себя комфортнее и не задумываться о необходимости приносить с собой наличные деньги в храм.

Электронная кружка для пожертвований

Возможности кружки

  • Сумма пожертвования на выбор
  • Оплата картой или мобильным устройством
  • Доступ ко всей статистике в личном кабинете

Варианты установки

  • Устанавливается в церковной лавке
  • Размещается на отдельной стойке в любом месте храма
  • Крепится к стене

Преимущества

  • Не требует администратора для управления и контроля
  • Разработана специально для нужд храма без лишних кнопок и элементов
  • Не требует открытия нового расчетного счета в банке
